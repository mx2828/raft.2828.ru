ТРЭКС / TREKS

Компания ТРЭКС / TREKS работает на Российском рынке с 2000 года, производит снаряжение для туризма и спорта:<>

  • туристические рюкзаки,<>
  • спальные мешки,<>
  • тенты групповые,<>
  • чехлы для лыж,<>
  • спортивные маты<>
  • и многое другое.<>

Для водного туризма Трекс изготавливает:<>

  • сплавные катамараны
    • Стерх (горбатый, с встроенными банками),<>
    • Лонгот (облегченный Иркут),<>
    • Иркут (самый дешевый сплавной катмаран на рынке),<>
    • Салга различных модификаций (уникальный катмаран-байдака),<>
    <>
  • рафты (экспедиционные лодки для сплава),<>
  • гермомешки различного объема,<>
  • спасательные жилеты (все жилеты сертифицированы),<>
  • современные оболочки ПВХ для байдарок “Таймень” и “Салют”.<>

Компания Трекс также занимается разработкой и производством чехлов различного назначения, упаковочных и транспортных сумок для приборов и оборудования, производит ТВЧ сварку различных тканей с полимерными покрытиями.<>

Среди клиентов ООО «ТРЕКС»: ОАО НПО «Сатурн» (чехлы для проекта SaM 146, ГТД 110 и другие), ОАО «Русская Механика»(чехлы и аксессуары для снегоходов «Буран», «Тайга»), «Смольский Водолазные Скафандры» (сумки для профессионального снаряжения) и другие.<>
