Сколько стоит катамарна для сплава

Стоимость поплавков (баллонов в шкуре) катамарана начинается с 8000 рублей за комплект. Катамараны отличаются комплектацией, конструкцией и материалами.<>

Конструкция катмаарана для сплава

Наибольшей популярностью пользуются двухслойные конструкции поплавков, где оболочка (шкура) изготовлена из ПВХ, а воздуходержащие камеры из других материалов. Для еще большего облегчения веса катамарана - верх оболочки попловка делается из синтетстических тканей, типа авизент, кордура. Баллоны, выпускаются в трёх модификациях:<>

  1. ЭКСТРА (ПУ, полиуритан) - баллоны из ПУ-ткани типа карингтон, ламкотек; самые современные, досточно легкие и неприхотливые баллоны для катамаранов любой конструкции.<>
  2. СТАНДАРТ - баллоны из прорезинниной ткани; классические баллоны оптимально по цене-качеству.<>
  3. ЛАЙТ (ПА, полиамид) - сверхлегкие и самые дешевые баллоны из ПА-пленки; не рекомендуется для серьезных походов или частых сборок. Рекомендуется, использовать,как временные, на один сезон или когда очень важен вес и/или цена катамарана!<>
