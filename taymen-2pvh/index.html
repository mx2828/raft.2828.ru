таймень 2 шкура

Оболочка (шкура) для байдарки Таймень 2 (3) изготовлена из ПВХ на тканевой основе ("Теза"). Плотность ткани на деке 700 г/кв.м. и 850 г/кв.м на днище. Шкура для байдарки усиленна по кильсону и по всем стрингерам ПВХ профилем (ширина профиля 52 мм, толщина профиля 2.2 мм). Масса оболочки 8.9 кг для двухместной модели и 10.2 кг для трехместной модели.
