Надувной рафт Енисей — Stream®

Все модели рафтов Енисей предназначены для походов по бурным рекам. Основной баллон рафта изготовлен из современных кордов материалов ПВХ и, для большей безопасности, разделён воздухонепроницаемыми перегородками на четыре отсека с отдельными клапанами. Съемное надувное дно с предохранительным клапаном для автоматического стравливания излишнего давления.

Три подушки-сидения, наружный леер, ручки подъема и переноса, съёмное дно, специальная упаковка (сумка), помпа, ремкомплект.
