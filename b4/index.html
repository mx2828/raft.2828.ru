Четырёхместные байдарки, надувные, каркасные, каркасно-надувные

Четырёхместные байдарки, как правило только надувные байдарки, типа Волга-Волга 3, Щука-4 и Налим-375, Налим-400 (снят с производства, выпускается только под заказ).<>

Из каркасных байдарок, только Таймень-3 можно использовать, как четырех местную байдарку, для удобства рекомендуется приобрести специальный фартук с 4-ми отсеками - ФТ34.<>

В основном же разумный предел, для байдарок - 3 места, далее, для сплава уже применяют катамараны и рафты, у которых практически неограниченная грузоподъемность и вместимость - по несколько десятков человек...<>
