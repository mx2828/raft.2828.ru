Палуба межбаллонная для катамарана

Межбаллонная палуба на маршрутах высших категорий недопустима, поскольку при оверкиле (перевороте) она может помешать экипажу выплыть.<>

На маршрутах средней сложности или при шторме допускается только сетчатая (не цельнотканевая) палуба, поскольку она пропускает сквозь себя водяные валы, обрушивающиеся на катамаран.<>

В остальных случаях можно использовать цельнотканевую палубу.<>

Палуба растягивается поперек и вдоль шнуровкой на раме катамарана или на петлях змейки. У катамаранов "Сталкер" основное натяжение палубы - продольное (поперечная шнуровка нужна лишь для перекрытия щелей между палубой и поплавками).<>

Палубы "Сталкер" - цельнотканевые (из ТЕЗЫ). Посередине (вдоль) проходит стропа, сквозь которую может стекать попавшая на палубу вода. Для ускорения стекания воды допускается пробивание дополнительных небольших отверстий в ТЕЗЕ.<>

Палубы "Сталкер" рассчитаны на растяжку между дополнительными (носовой и кормовой) поперечинами (см. п. "Рамы" выше). Однако, если пользователь не хочет устанавливать дополнительные поперечины, можно установить палубу на основных поперечинах. Для этого палуба должна быть короче: для к4 - палуба к2, а для к6 - палуба к4 или к2.<>

Для установки мачты необходимо прорезать в центральной стропе палубы "Сталкер" отверстие-щель. Чтобы укрепить концы щели, на стропу в некоторых местах нашиты специальные поперечные закрепки.<>
