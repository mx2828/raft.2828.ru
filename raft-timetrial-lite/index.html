ТРИТОН-460 ТРИТОН-460L

8-местный рафт "Тритон-460" выпускается в 2 модификациях:

  • Тритон-460 / TRITON-460
  • Тритон-460 Light / TRITON-460L

Данный рафт предназначен для походов по сложным бурным рекам.

В конструкцию рафта для большей безопасности сплава : бортовой баллон имеет 2 независимые секции.

Для большего комфорта и безопасности рафты оснащаются удоьными и прочными ручками для подъема и переноса; дно обоих рафтов оснащено специальными приспособлениями для крепления груза.
